Tube Server Manager - API

  • Description:
    1. A API permite gerenciar e listar mídias do YouTube, como vídeos, séries e músicas.
    2. Projetada para simular o comportamento de uma aplicação em produção, utilizando dados mockados para fins de desenvolvimento e teste.
    3. Os dados mockados foram armazenados em um arquivo chamado data.json, que simula respostas de um banco de dados real.

    • Routes

    Somente o método GET está disponível para usuários não autenticados, permitindo consultas públicas aos dados.

    • GET /api/v1/media
    • GET /api/v1/media/1
    • GET /api/v1/media?category=terror
    • GET /api/v1/media?terms=velozes
    • POST /api/v1/media
    • PUT /api/v1/media/1
    • PATCH /api/v1/media/1
    • DELETE /api/v1/media/1
